##	Git
###	什麼是 Git?
*	Git是一個分佈式版本管理系統，是為了更好地管理Linux內核開發而創立的。
*	Git可以在任何時間點，把文檔的狀態作為更新記錄保存起來。因此可以把編輯過的文檔復原到以前的狀態，也可以顯示編輯前後的內容差異。
*	而且，編輯舊文件後，試圖覆蓋較新的文件的時候（即上傳文件到服務器時），系統會發出警告，因此可以避免在無意中覆蓋了他人的編輯內容。
